Run Basile

Retour vers les projets

Année de création :
2016

Nombre de joueur :
2-4

Plateforme :
PC

Genre : Combat

Durée du projet : 1 semaine

Taille de l'équipe : 4 personnes

Resumé : Quatre frères ​chasseurs s'affrontent pour décider lequel d'entre eux aura l'honneur de tuer Basile le Lapin.

Gameplay : Courez, sautez, giflez ou tirez sur vos adversaires dans deux arènes différentes.

Mon role : Durant ce projet, j'étais l'un des Game Designers et le Lead Programmer.

Ce que j'ai réalisé :

  • Designé le coeur du mode Combat
  • Conceptualisé deux autres modes de jeu, avec des contrôles similaires au mode Combat
  • Programmé la partie Combat
  • Equilibré l'expérience pour que le jeu reste amusant avec deux joueurs comme avec quatre, tout en prototypant le jeu​
